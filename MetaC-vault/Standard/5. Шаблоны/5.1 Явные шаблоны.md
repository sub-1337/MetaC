В языке есть явные шаблоны.
В угловых скобках указывается передаваемый в шаблон аргумент.

Пример использования с стандартным шаблоном массива.
```
import array;

let mut arr : array<int>;
arr.append(10);
arr.append(30);

println!(arr); // 10 30
```

Параметр шаблона может быть:
1) int (целое число)
2) тип
3) bool
