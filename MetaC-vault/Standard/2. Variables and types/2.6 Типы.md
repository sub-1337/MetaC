### type()
В языке можно брать тип от переменных.

```
let a : int = 10;
let aType : type = type(a);
let b : aType = 20;

println!("{} {}", a, b);
// 10 20
```

### sizeof()
Можно брать размер переменной, если это возможно.

```
let a : int64;
let sz = sizeof(a);
println!(sz); // 4
```
### sizeof::bite()
Можно также брать размер переменной в битах (только для int).
```
let a : int64;
let sz = sizeof::bite(a);
println!(sz); // 64
```
