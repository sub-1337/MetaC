В языке есть перечисления.

```
enum A
{
	a,
	b,
	c,
	d,
	e
}

let t1 = A::a;
let t2 = A::e;
```

### Суммирование перечислений
```
enum A
{
	a,
	b
}
enum B
{
	c,
	d
}
```
Мы можем совмещать несколько перечислений так:
```
let multipleEnum : A + B;
```
Или так:
```
let C : type = A + B;
let multipleEnum : C;
```
Или так:
 ```
 enum C(A + B);
 ```
В таком случае объект multipleEnum будет содержать **одно**
значения из A или B. Например `a` или `d`.


### Перечисления на основе типа
```
enum State(bool)
{
	on = true,
	off = false
}

enum Severity(int)
{
	trace = 0,
	debug = 1,
	
}
```
