**Расширение исходных файлов** `.mc` - meta C
Это файлы которые содержат исходный код.

**Расширение экспортных файлов** `.mh` - meta Header
Эти файлы содержат экспортируемые объявления, 
они используются вместо `.h` файлов в C/C++, 
с той разницей, что они не предназначены для изменения, 
а идут вместе с бинарными файлами после компиляции кода.
Вам больше не нужно менять код в 2х местах для поддержки 
изменения в коде.

**Бинарные файлы** `.mb` - meta Binary
Это файлы с объектным кодом, которые можно простроить заранее.

**Файл постройки** `MetaMake.txt` /  `MetaMake.mc`
Эти файлы используются для конфигурирования процесса постройки.
В нём можно описать список используемых библиотек, путей и
описать какие файлы участвуют в постройке.
