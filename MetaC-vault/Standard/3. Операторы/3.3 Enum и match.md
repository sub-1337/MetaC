
Можно определить несколько вариантов на одно действие.
`case 0,2,3 => ...`
### Пример с больше меньше
```
let const x = input!(int);

match(x)
{
	< 0 => println!("Меньше нуля");
	0 => println!("Ноль");
	> 0 => println!("Больше");
}
```


### Пример с диапазоном
```
let const x = input!(int);

match(x)
{
	[-10..0) => println!("1");
	0 => println!("2");
	[0..10], [20..30) => println!("3");
	default => println!("4");
}
```
