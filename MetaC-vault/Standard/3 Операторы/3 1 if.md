Оператор if и многие другие операторы возвращают значение как в rust.

```
let x = 10;
if (x == 10)
{
	println!("equal");
}
else
{
	println!("not eqrual");
}
```
Пример использования.

```
let res : int = if (str == "test") {10} else {30};
```
Присвоить переменной значение 10 если равенство верно
и 30 если нет.
