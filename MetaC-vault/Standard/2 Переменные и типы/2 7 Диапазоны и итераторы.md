### Сумма-диапазон
`[0,2,3]` - диапазон куда входит 0 и 2 и 3
Может быть приведён к типу контейнера.

### Диапазоны граничные
`[0..5]` От 0 до 4 включительно.
`[0..=5]` От 0 до 5 включительно

`[!0..5]` содержит 1,2,3,4
`[!0..=5]` содержит 1,2,3,4,5
### Суммы
`[0..3, 5, 10]` содержит 0,1,2,5,10
`[0..=2] + [4,6]` содержит 0, 1, 2, 4, 6

### Сохранение диапазона
```
let const rangeSaved : type::range = [0..5];
```

### Модификаторы

Модификаторы:
* `filter` - отфильтровать элементы функцией
* `transform` - применить функцию к элементам
* `take` - первые элементы
* `drop` - пропустить элементы
* `reverse` - разворот
* `keys` / `values` - для мапы
* `split` - для строки или массива
*  `iota` - аналог стандартной записи
*  `|` - объединение модификаторов
*  `sort` - сортировка
*  `chunk` - сгруппировать элементы
*  `slide` - скользящее окно
*  `zip` - объединение
*  `repeat` - повторить
