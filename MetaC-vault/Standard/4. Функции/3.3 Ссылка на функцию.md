На функцию можно создавать ссылки.
Можно создать общую ссылку - тогда она будет автоматически 
подставлять в имя функции расширение имени, 
а можно создать указатель на конкретную функцию.

```
fn func::test(let const a) -> ()
{
	println!("Test");
}
fn func::test2() -> ()
{
	println!("Test 2");
}

let const fref : type::fn = func;

fref(10); // Test
fref();   // Test 2

let const freffull : type::fn = func::test2;
freffull(); // Test 2
```
