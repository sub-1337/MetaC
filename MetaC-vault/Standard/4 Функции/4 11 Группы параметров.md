```
fn func(let a, let b)
{
	print(a,b);
}
fn funcT1(let tuple t)
{
	assert(t.len == 2);
	func(t[0], t[1]);
}
fn funcT2(let tuple t)
{
	func(@t);
}

funcT1(10, "aaa"); // 10 aaa
funcT2(10, "aaa"); // 10 aaa
```
Мы можем передать переменное количество параметров 
в функцию.
При этом мы можем работать с массивом параметров
как с тьюплом.

Внимание!
`let a : tuple` не то же самое что `let tuple a`
