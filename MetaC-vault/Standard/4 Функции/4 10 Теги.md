

```
fn func(let tag t1, let tag t2)
{
	print(" {t1} {t2} ");
}

func(#t1, #t2); // true true
func(#t2, #t1); // true true
func(#t1);      // true false
func(#t2);      // false true
func();         // false false
```
Ключевое слово `tag` в объявлении говорит что
это опциональный параметр-тег. При передаче в функцию
тег может стоять в любой позиции.
При вызове функции теги можно указать в любой позиции.
```
fn func(let a, let tag t, let b)
{
	print("{a} {t} {b}");
}

func(10,50);       // 10 false 50
func(10, #t , 60); // 10 true 60
func(#t, 20, 30);  // 20 true 30
```

При этом тег не будет считаться за обычный параметр,
не будет влиять на позицию других параметров.

```
fn func(let tag t : int)
{
	print(t);
}

func(#t := 10);     // 10
```
Тег может содержать значение.
