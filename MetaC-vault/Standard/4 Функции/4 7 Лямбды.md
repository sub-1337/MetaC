
### Сокращённые лямбды
Мы можем объявить лямбду в сокращённом виде.

Например так: `fn::lam {>}`
Что эквивалентно `fn (let a, let b) { return a > b; }`

Или `fn::lam {<=2}`
Что эквивалентно `fn (let a) { return a <= 2; } 

Или `fn::lam {in [0..2]}`
Что эквивалентно `fn (let a) { return a >= 0 and a < 2; }`
