Перегрузить получаемые значения можно разными способами.
Сделать это в стиле c++ можно так:

```
fn function(let const a : str)
{
    println!("String {}", a);
}

fn function(let const a : int)
{
    println!("Integer {}", a);
}
```

Если для вызова доступно несколько функций вызовется самая специализированная. Если и таких несколько вызовется последняя.
